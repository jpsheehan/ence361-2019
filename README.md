## Hardware Setup:

### Test Rig:

| **Wire Color** | **Purpose**   | **Header Pin** | **Jumper Pin** |
| -------------- | ------------- | -------------- | -------------- |
| Black          | Ground        | GND            | GND            |
| White          | Altitude      | PE4            | J1-05          |
| Purple         | Yaw (Phase A) | PB0            | J1-03          |
| Grey           | Yaw (Phase B) | PB1            | J1-04          |
